    CPEP 311 · Midterm Project

    16 Segment Display Logic in Circuits

    A hardware logic circuit project that drives a 16-segment display using combinational logic. An 8-bit input via a 3-bit DIP switch is decoded through a network of logic gates to control each of the 16 segments — displaying characters and digits without any microcontroller.

    Components Used

    3-bit DIP Switch NOT Gate 3× OR Gate 5× AND Gate 2× XOR Gate 16 Segment Display 330Ω Resistor Breadboard 22AWG Solid Wire

    Robotics Project · PD + Kalman Filter

    Self-Balancing Robot

    A two-wheeled self-balancing robot that uses a PD (Proportional-Derivative) control algorithm combined with a Kalman Filter for accurate angle estimation. The MPU6050 IMU sensor continuously reads tilt data, which is processed by the Arduino Nano to drive the motors and keep the robot upright in real time.

    Components Used

    Arduino Nano MPU6050 IMU Motor Driver 2× Geared Motor 2× Tires 4× 18650 Battery Illuminated Switch Small Breadboard

    Algorithm

    PD Control Kalman Filter
